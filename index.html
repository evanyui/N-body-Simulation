Instructions:
"Q" to enter insert mode to enable setting mass size by using number keys upon creation (without typing on textbox).
Press "Q" again to exit insert mode or click on screen to create particle then exit.
"C" to clear.
"Z" to enable collision.

Note: Not to physical scale. Mass is size * factor(1). Calculation is based on momentum, centripetal force, and universal gravitational force. Taken constant G as 1.
